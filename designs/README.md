# ZIPboost Dark Blue/Purple Designs

All designs feature dark blue/purple color schemes with different styles.

## To Switch

```bash
cp designs/neon-cyber/style.css public/style.css
# Reload extension
```

---

## Available Designs

### 1. Glassmorphism - `/glassmorphism`
Frosted glass effect with blur and gradients.
- Purple glow accents
- Translucent panels
- Gradient background

### 2. Midnight Clean - `/midnight-clean`
Clean, minimal design with navy/indigo tones.
- No heavy effects
- Simple underline tabs
- Solid colors

### 3. Aurora Gradient - `/aurora-gradient`
Deep purple/blue gradient background.
- Aurora-like feel
- Subtle blur effects
- Soft glowing buttons

### 4. Neon Cyber - `/neon-cyber`
Cyberpunk-inspired with neon glowing accents.
- Vibrant purple/cyan neon
- Glow effects on hover
- Sharp, techy feel

### 5. Soft Depths - `/soft-depths`
Muted, calm dark with soft blue undertones.
- Relaxed, gentle feel
- Subtle rounded corners
- Soft shadows

---

All have dark + light themes.
